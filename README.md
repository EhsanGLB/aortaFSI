# aortaFSI
This is a case to model a aorta using FSI and STL file with cfMesh written based on foam-extend-4.1 and solids4foam


## Installation
It is working on foam-extend-4.1.
```bash
git clone https://github.com/EhsanGLB/aortaFSI.git
```


## Getting Started
```bash
./Allrun
```
